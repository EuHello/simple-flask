# Objective
Learn to create a quick API with Flask.

## Run
`flask --app flaskr run --debug`  

## HTTP methods
1. GET /hello
2. GET /model/view
3. POST /model/predict


## Note: 
if using POSTMAN, to prevent JSON format error, ensure 

1. Headers: Content-Type = application/json
2. Sample JSON payload
{
  "Age": "200"
}
